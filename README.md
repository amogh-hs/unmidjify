# unmidjify

A Clojure tool that helps in converting a Midje based test codebase to a clojure.test based codebase.

## Installation

Download from http://example.com/FIXME.

## Usage

In the REPl,

(require 'unmidjify.core),

(unmidjify <path/to/file/or/directory>)
